# Nasdaq-volatility-var-backtest
This project analyzes daily NASDAQ returns using several time-series models and evaluates their performance in forecasting distributions and measuring market risk. The study compares linear autoregressive models, threshold autoregression, and volatility models with conditional heteroskedasticity.

The goal is to examine whether modeling conditional volatility improves tail-risk estimation and risk management metrics such as Value-at-Risk (VaR) and Expected Shortfall (ES).
